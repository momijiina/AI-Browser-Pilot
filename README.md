# AI Browser Pilot
LM Studio を使って AI がブラウザを自律操作する Chrome 拡張機能です。  
自然言語でタスクを指示するだけで、AI がページの要素を認識し、クリック・入力・ナビゲーションなどを自動で実行します。

テスト中ですが理論的にはログインページにもログイン可能<br/>
標準ブラウザの為、ボット検出も回避(これが目的で作成してます)

## 必要なもの

- **Google Chrome系**
- **LM Studio**（https://lmstudio.ai）
- 4B パラメータ以上の LLM モデル（推奨:Qwen3.5 4B）
  - 0.8B などの小型モデルでは JSON 出力が不安定な為(Qwen3.5ならある程度？)

## インストール手順

### 1. LM Studio のセットアップ

1. [LM Studio](https://lmstudio.ai) をダウンロード・インストール
2. LM Studio を起動し、モデルをダウンロード（4B 以上推奨）
3. モデルをロードし、ローカルサーバーを起動（デフォルトポート: `1234`）

### 2. 拡張機能のインストール

1. Chrome で `chrome://extensions` を開く
2. 右上の **「デベロッパーモード」** を有効にする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. この拡張機能のフォルダを選択
5. ツールバーに拡張機能のアイコンが表示されれば完了

### タスクの実行

1. 設定ポップアップの **「🤖 サイドパネルを開く」** をクリック
2. サイドパネル下部のテキストエリアにタスクを入力
3. **▶ ボタン**を押す（または Enter キー）で実行開始
4. AI の思考・操作・結果がリアルタイムでログに表示される
5. 途中で止めたい場合は **⏹ ボタン** をクリック

### タスクの入力例

| 入力例 | AI の動作 |
|--------|----------|
| `Googleで猫の画像を検索して` | Google で「猫の画像」を検索 |
| `下にスクロールして` | ページを下にスクロール |
| `このページのタイトルを教えて` | ページ情報を読み取って回答 |
| `GitHubにログインして` | GitHub のログインフォームを操作 |
| `戻るボタンを押して` | ブラウザの戻るを実行 |
